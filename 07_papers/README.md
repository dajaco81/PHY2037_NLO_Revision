## How to use

**Start here:** Read `EXAM_ANALYSIS.md` — it maps which topics appear every year (certain) vs. sometimes (likely) vs. rarely.

**Then:** Work through `PAST_PAPER_SOLUTIONS.md` for question types not in `05_exam_prep/MODEL_QUESTIONS.md`. The model questions file has been updated (Q13 onwards) with new question types extracted from the real papers.

**Exam format:** 4 questions, answer ALL, 2 hours, closed note, physical constants sheet provided.
