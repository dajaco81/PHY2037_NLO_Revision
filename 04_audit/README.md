# 04_audit — Quality & Consistency Audit

This directory contains files documenting notation choices, identified inconsistencies across lectures, and suspected transcript errors.

## Files

| File | Purpose |
|------|---------|
| `NOTATION_STANDARDIZATION.md` | Canonical symbol choices used throughout this repository |
| `INCONSISTENCY_LOG.md` | Contradictions or variations between lectures |
| `POSSIBLE_TRANSCRIPT_ERRORS.md` | Statements that appear garbled or physically incorrect |

## Usage

Check `NOTATION_STANDARDIZATION.md` before exam to ensure you are using consistent symbols. The other files are for completeness and verification only.
