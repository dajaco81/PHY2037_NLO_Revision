# 03_master — Module-Level Synthesis

This directory contains the top-level revision documents synthesising content across all lectures.

## Files

| File | Purpose |
|------|---------|
| `MASTER_REVISION_MAP.md` | Complete topic-by-topic roadmap of the module |
| `EQUATION_SHEET.md` | Every examinable equation, numbered and contextualised |
| `DERIVATION_BANK.md` | Step-by-step derivations for all flagged examinable derivations |

## How to Use

Start with `MASTER_REVISION_MAP.md` to understand the overall structure. Then work through `EQUATION_SHEET.md` and `DERIVATION_BANK.md` systematically. Use `CONCEPT_MAP.md` to check understanding of connections.
