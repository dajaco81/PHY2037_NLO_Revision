## Module Topic Map

The module progresses through nine broad topics:

1. **Introduction** (L01–L02): Course overview, prerequisite physics recap (EM, harmonic motion, QM, Gaussian/Fourier maths)
2. **Optical Imaging** (L03–L04): Contrast mechanisms, fluorescence, spatial resolution, confocal microscopy, scattering
3. **Linear Light–Matter Interaction** (L05–L07): Lorentz oscillator model, susceptibility χ(ω), vibrational spectroscopy (IR & Raman)
4. **Nonlinear Susceptibilities** (L08–L09): Power series of P, χ₂ and χ₃ estimates, symmetry, perturbation theory, SHG derivation
5. **Wave Equation & Phase Matching** (L10–L11): Nonlinear wave equation, coupled amplitude equations, phase matching schemes
6. **Laser Pulses** (L12, L14–L15): Peak intensity, pulse energy/power, time-bandwidth product, mode locking, pulse formation
7. **Nonlinear Optical Microscopy** (L14–L16): 2PEF, SHG/THG, CARS, SRS — principles and advantages
8. **Laser Sources** (L17): Ultrafast laser design, mode locking (SESAM/KLM), OPO, frequency conversion
9. **Applications** (L18): Cancer diagnostics, drug delivery, nanoparticle imaging, microplastics

## Using This Repository for Exam Preparation

The May exam tests understanding across the full module. Key areas repeatedly flagged by the lecturer as examinable:

- Deriving χ from the Lorentz oscillator (second-order)
- SHG derivation using perturbation theory
- Phase matching condition and sinc² intensity profile
- Pulse parameters (energy, peak power, peak intensity)
- Time-bandwidth product derivation and calculation
- Confocal microscopy principle and limitations
- Nonlinear microscopy advantages (intrinsic confinement, depth penetration, resolution squeeze)
- Raman vs IR selection rules
- CARS wavelength calculation
