# L17 Digest — Laser Sources: Ultrafast Lasers and Frequency Conversion

## Core Concepts

1. **Cavity mode spacing:** Δν = c/2L (modes quantised by cavity length L; only modes within gain bandwidth lase).

2. **Mode locking:** Force all modes to have the same phase → pulses form by constructive interference at one point in the cavity per round trip. Repetition rate = c/2L.

3. **SESAM (saturatable absorber):** Low intensity → absorber saturates slowly → high loss. High intensity pulse → absorber saturates instantly → low loss. Favours transmission of high-intensity packets. Relaxation time ~ps → suitable for ps pulses (needed for CARS/SRS).

4. **KLM (Kerr lens mode locking):** Intensity-dependent refractive index n = n₀ + n₂I creates a lens for high-intensity beams. Aperture selects the self-focused (high intensity) beam. Responds essentially instantaneously → suitable for 100 fs pulses.

5. **Dispersion compensation:** GVD stretches pulses in every optical element. Compensated by grating or prism pairs inside the cavity (gives blue light a shorter path).

6. **OPO:** Difference frequency generation in a resonant cavity. Input: pump (ω_P) → output: signal (ω_S) + idler (ω_i) with ω_P = ω_S + ω_i. Temperature-tunable → adjustable output wavelength.

## Physical Intuition

**Why mode locking gives pulses:** Imagine all cavity modes (at slightly different frequencies) being forced into the same phase. At the moment they all line up, they add constructively → a bright short pulse. A moment later they drift slightly out of phase → destructive interference → dark. The repetition rate is just how long it takes light to go around the cavity: τ = 2L/c → f_rep = c/2L.

**SESAM:** A real semiconductor absorber with a certain maximum absorption cross-section. Once all the available electrons are excited to the conduction band, no more photons can be absorbed → the material is "bleached" and becomes transparent. High-intensity pulses bleach it instantly; low-intensity CW light bleaches it slowly (lower effective transparency).

**Kerr lens:** Like a gradient-index lens but created by the intensity profile of the beam itself. The Gaussian beam creates a parabolic index profile → focuses itself. A hard aperture then acts like a pinhole that only the self-focused (high-intensity) beam passes through.

**OPO:** One energetic pump photon splits into two lower-energy photons (signal and idler). The crystal and temperature determine which wavelengths are generated (phase matching condition). Cavity mirrors reflect the signal wavelength, allowing it to build up over many passes.

## Exam Relevance

- Be able to explain how SESAM mode locking works (draw the loss vs. intensity curve)
- Be able to explain how KLM works (Kerr effect → self-focusing → aperture selects pulses)
- Know which method gives fs vs. ps pulses and why (SESAM: ps due to relaxation time; KLM: fs, near-instantaneous)
- **[RED BOX]** OPO wavelength equation: 1/λ_P = 1/λ_S + 1/λ_i

## Common Mistakes

1. Saying SESAM produces femtosecond pulses (it produces picosecond — timescale determined by semiconductor relaxation)
2. Saying KLM uses a separate element (in Ti:Sapphire the crystal itself is the Kerr medium)
3. Forgetting the OPO cavity — DFG alone is single-pass; OPO is resonant
4. Using ω_P = ω_S − ω_i instead of ω_P = ω_S + ω_i (both positive)
