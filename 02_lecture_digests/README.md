# 02_lecture_digests — Lecture Digests

One structured digest per lecture, synthesising the key content from the cleaned transcripts.

## File Naming

`L{number}_{YYYY-MM-DD}_digest.md`

## Digest Structure

Each digest contains:

1. **Core Concepts** — the main physics ideas introduced
2. **Key Equations** — numbered and labelled
3. **Definitions** — precise definitions of new terms
4. **Derivations** — step-by-step where the lecturer derived something
5. **Physical Intuition** — qualitative explanations and analogies
6. **Worked Examples** — numerical or algebraic examples from the lecture
7. **Exam Relevance** — explicitly examinable content
8. **Lecturer Exam Hints** — red box items and explicit exam flags
9. **Common Mistakes** — warned about in lecture
10. **Transcript Ambiguities** — unclear or potentially mis-transcribed content
11. **Links to Related Lectures** — cross-references

## Usage

Use these digests when a concept in the master or exam-prep files needs deeper context. Do **not** read them cover-to-cover for revision — use the master files in `03_master/` first.
