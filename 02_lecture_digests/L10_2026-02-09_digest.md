# L10 Digest — Nonlinear Wave Equation and Phase Matching

## Core Concepts

1. **Dispersion:** n(ω) increases with ω for normal dispersion. This means k = nω/c is not simply proportional to ω → k vectors at different frequencies do not automatically satisfy k₁+k₂=k₃.

2. **Nonlinear wave equation:** Derived from Maxwell's equations by separating linear and nonlinear polarisation. The NL polarisation P_NL acts as a **source term** on the right-hand side, driving new frequency components.

3. **Phase matching condition:** For efficient NLO signal generation, the wave vectors must satisfy:
   $$k_3 = k_1 + k_2 \quad (\Delta k = 0)$$

4. **Signal scales as N²** when phase-matched (amplitudes from all N atoms add coherently → signal × N; intensity × N²).

5. **Signal intensity ∝ sinc²(ΔkL/2):** With phase mismatch Δk, the signal oscillates rather than growing — maximum at Δk=0, first zero at ΔkL/2 = π.

6. **Coupled amplitude equations:** Derived from SVEA applied to NL wave equation — show how field amplitudes at different frequencies exchange energy as they propagate.

## Key Equations

| Equation | Name |
|----------|------|
| k = nω/c | Wave vector (with dispersion) |
| Δk = k₃ − k₁ − k₂ | Phase mismatch |
| ∇²E − (n²/c²)∂²E/∂t² = (1/ε₀c²)∂²P_NL/∂t² | Nonlinear wave equation |
| dA₃/dz = (iω₃d_eff/n₃c)A₁A₂e^{iΔkz} | Coupled amplitude equation (SFG) |
| I₃ ∝ L² sinc²(ΔkL/2) | Signal intensity vs phase mismatch |
| L_c = π/Δk | Coherence length (first zero of sinc²) |

## Derivations

### Nonlinear Wave Equation

> **[RED BOX — derivation]**

Starting from Maxwell's equation ∇ × **H** = ∂**D**/∂t:
1. Take curl: ∇ × (∇ × **E**) = −μ₀∂/∂t(∇ × **H**)
2. Use vector identity: ∇ × (∇ × **E**) = ∇(∇·**E**) − ∇²**E**
3. Substitute **D** = ε₀**E** + **P** = ε₀(1+χ)**E** + **P_NL** → separates linear (→ modifies ε, gives n²) from NL
4. Make ∇·**E** ≈ 0 (no free charges; small correction)
5. Result: ∇²**E** − (n²/c²)∂²**E**/∂t² = (1/ε₀c²)∂²**P_NL**/∂t²

### Coupled Amplitude Equation (SFG)

> **[RED BOX — derive or reproduce]**

1. Write E_j = A_j(z)exp[i(k_j z − ω_j t)] + c.c. where A_j(z) is slowly varying
2. Substitute into NL wave equation
3. **Key step:** ∂²/∂z²[A_j(z)e^{ik_j z}] = (∂²A_j/∂z² + 2ik_j ∂A_j/∂z − k_j²A_j)e^{ik_j z}
4. SVEA: |∂²A_j/∂z²| ≪ |k_j ∂A_j/∂z| → drop first term
5. The k_j² terms cancel with the left-hand side n²ω_j²/c² terms
6. Result: dA₃/dz = (iω₃d_eff/n₃c) A₁A₂ e^{iΔkz}

### Signal Intensity

Integrate dA₃/dz with A₁, A₂ constant (undepleted pump):

$$A_3(L) \propto \int_0^L e^{i\Delta k z}dz = \frac{e^{i\Delta kL} - 1}{i\Delta k} = Le^{i\Delta kL/2}\text{sinc}\left(\frac{\Delta kL}{2}\right)$$

$$I_3 = |A_3|^2 \propto L^2\text{sinc}^2\left(\frac{\Delta kL}{2}\right)$$

## Physical Intuition

Think of N atoms as N little antennas all oscillating and emitting. For the signal to build up, each atom must emit in phase with all the others *at the detection point*. If the phase velocity of the driving fields (combined k₁+k₂) doesn't match the phase velocity of the signal (k₃), the atoms get out of phase with each other as z increases — constructive then destructive interference → oscillating (not growing) signal.

With perfect phase matching: all atoms emit in phase → amplitudes add → signal ∝ N → intensity ∝ N².

## Common Mistakes

1. Forgetting A_j(z) depends on z when differentiating → missing the product rule term
2. Getting the sinc argument wrong — it's ΔkL/2, not ΔkL
3. Saying signal goes to zero at ΔkL = π (wrong) — it's ΔkL/2 = π → ΔkL = 2π
4. Forgetting the L² prefactor — intensity grows quadratically with interaction length when phase matched
