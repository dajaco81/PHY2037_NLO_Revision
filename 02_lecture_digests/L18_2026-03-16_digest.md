# L18 Digest — Applications of Nonlinear Optical Microscopy

## Core Concepts

1. **NLO advantages summary (exam critical):**
   - No pinhole needed (intrinsic confinement, signal only at focus)
   - Deeper penetration (2× excitation wavelength → 16× less scattering; emitted signal scattered freely)
   - Resolution partially recovered (nonlinear squeeze: 1/√n factor)

2. **Forward vs. backward CARS (phase matching in application):**
   - Forward: bulk material phase matches (Δk × L < π) → strong bulk signal
   - Backward (EPI): k₃ reversed → large Δk → only sub-wavelength objects give backward CARS
   - Application: suppress bulk cell background; image nanoparticles selectively

3. **Virtual H&E:** SRS images at CH₂ and amide I bands → maps lipid and protein → pseudo-colour → matches H&E appearance. AI trained on historical slides can diagnose from virtual H&E in real time.

4. **Deuterium labelling:** Replace H with D in drug molecule → shifts C–H stretch (~2950 cm⁻¹) to C–D stretch (~2100 cm⁻¹) in the "Raman silent region". Chemically unchanged but now uniquely detectable.

## Key Points

| Application | NLO technique | What is imaged |
|------------|-------------|---------------|
| Nanoparticle uptake | EPI-CARS | Sub-λ particles in cells |
| Cancer diagnosis | SRS virtual H&E + AI | Unstained tissue |
| Drug skin penetration | SRS (C–H, C–D, silent region) | Drug + solvent diffusion |

| Myosin in muscle | SHG | Muscle disease models |

## Forward vs. Backward CARS (Key Exam Point)

> **[RED BOX — connects phase matching to applications]**

**Forward:** k₃ in same direction as k₁, k₂. Δk = k₃ − 2k₁ ≈ small. Phase matching condition satisfied for L ≥ λ (bulk material). **Strong signal from bulk.**

**Backward (EPI):** k₃ now points backwards → Δk_eff = k₃ + 2k₁ ≈ very large. Phase matching condition requires L < λ/2. **Only sub-wavelength objects give EPI signal.**

Physical insight: the contributions from the two halves of the focal volume cancel for bulk material in the backward direction (they're opposite phases). Only if an interface or sub-λ particle breaks this symmetry does a backward signal appear.

**Application:** Image nanoparticles (100–200 nm) inside cells — the particles give EPI signal; the bulk cells do not (suppressed by phase mismatch). Can overlay with SRS from cells to show uptake.

## Deuterium Labelling

> **[RED BOX — exam-relevant]**

**Why:** Drug molecules may have no unique Raman peak. Fluorescent labels are too large.

**How:** Synthesise the molecule with deuterium (D) replacing hydrogen (H). C–D stretch:

$$\nu_{C-D} = \frac{1}{2\pi}\sqrt{\frac{k}{\mu_{C-D}}}$$

Since m_D ≈ 2m_H: μ_r increases → ω_vib decreases → C–D stretch shifts to ~2100 cm⁻¹

**Raman silent region:** 1900–2800 cm⁻¹ has no endogenous biological Raman peaks → C–D signal is background-free.

**Chemical equivalence:** C–D and C–H bonds are chemically nearly identical (same electron configuration; mass difference is <1% of molecular weight). Minimal perturbation — unlike a fluorescent label which can double the molecular weight.

## Common Mistakes

1. Thinking EPI-CARS and forward CARS give the same image (they don't — EPI is selective for sub-λ)
2. Saying deuterium changes the chemistry of the molecule (it doesn't — only the vibrational frequency)
3. Forgetting that virtual H&E requires AI/machine learning to map SRS channels onto the H&E colour scheme
