# L11 Digest — Phase Matching Schemes

## Core Concepts

1. **Normal dispersion always gives Δk ≠ 0:** For SHG, perfect phase matching requires n(2ω) = n(ω) — impossible since n increases with ω.

2. **Three classical phase matching methods:**
   - **Angle tuning:** birefringent crystal, rotate to change extraordinary n
   - **Temperature tuning:** lithium niobate at ~140°C; tunes n via thermal birefringence
   - **Quasi-phase matching (periodic poling):** flip crystal orientation periodically so phase mismatch resets at each coherence length

3. **Gouy phase shift:** a focused beam acquires a total phase shift of **π radians** going through the focus. Each input beam contributes π/2.

4. **SHG in microscope:** total Gouy phase = π (from 2 input beams). Combined with Δk from dispersion, phase matching requires interaction length L < λ₁/2. This is just achievable → SHG works in a microscope.

5. **THG in microscope:** total Gouy phase = 3π/2 (from 3 input beams). Much larger Δk requirement → L < λ₁/3. Very hard to satisfy → THG only from interfaces and very thin objects.

## Key Points

| Process | Phase matching condition in tight focus | Result |
|---------|----------------------------------------|--------|
| SHG | L < λ₁/2 | Works, images collagen/myosin |
| THG | L < λ₁/3 | Interface-selective only |
| 4WM (2ω₁−ω₂) | Gouy phases partially cancel | Works from bulk |

## Definitions

- **Birefringence:** property of a material where refractive index depends on polarisation direction
- **Ordinary ray (o-ray):** polarisation perpendicular to optic axis; n_o is fixed
- **Extraordinary ray (e-ray):** polarisation with component along optic axis; n_e depends on angle θ
- **Walkoff:** spatial separation of ordinary and extraordinary beams when angle ≠ 0° or 90°
- **PPLN (periodically-poled lithium niobate):** crystal with alternating domain orientations, period matched to coherence length
- **Gouy phase shift:** phase shift of π total when a Gaussian beam passes through its focus

## Physical Intuition

**Quasi-phase matching:** Imagine pumping someone on a swing that has the wrong resonance — they keep swinging back before you push. If you could reverse the direction of your push exactly when the signal is about to reverse, you can maintain constructive buildup. That's what periodic poling does — it flips the sign of the coupling when the phase mismatch would otherwise cause the signal to decrease.

**Gouy phase:** A tightly focused beam has a range of k-vector directions (the cone of the NA). The beam has a slightly different phase velocity through the focus compared to a plane wave. This is a purely geometrical consequence of focusing.

**THG and interfaces:** The large Gouy + dispersion phase mismatch means that any contribution from one side of the focus (positive Δk) cancels with contribution from the other side (negative Δk) for bulk material. Only at an interface — where the material properties change within the focal volume — does the cancellation fail.

## Common Mistakes

1. Saying SHG doesn't work in a microscope (it does, just barely)
2. Forgetting the Gouy phase adds to Δk (it's an additional contribution, not a replacement)
3. Confusing quasi-phase matching with true phase matching (signal still oscillates, just less severely)
4. Not remembering that THG interface selectivity comes from the same sinc² / Gouy phase argument as SHG
