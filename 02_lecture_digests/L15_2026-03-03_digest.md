# L15 Digest — NLO Microscopy: Resolution, SHG, THG, and CARS Preview

## Core Concepts

1. **Nonlinear squeeze effect:** n-photon process scales as Iⁿ. Gaussian PSF squared → narrower Gaussian (σ → σ/√n). Resolution recovered:
   $$d_{nP} = \frac{d_{1P}}{\sqrt{n}} \quad \text{after accounting for increased excitation wavelength}$$

   Net resolution for 2PEF using dye excited at λ_ex with 2λ_ex excitation:
   $$d_{2P} = \frac{0.61 \times 2\lambda_{ex}}{\text{NA}} \times \frac{1}{\sqrt{2}} = \frac{0.61\sqrt{2}\lambda_{ex}}{\text{NA}}$$

2. **Parametric vs. non-parametric:** Parametric = no net energy transfer to material → coherent emission → phase matching critical. Non-parametric = energy deposited → incoherent emission → no phase matching needed.

3. **SHG in microscopy:** phase matching condition L < λ₁/2 (just satisfiable) → works for collagen and myosin (non-centrosymmetric biological structures).

4. **THG in microscopy:** L < λ₁/3 (too tight for bulk) → only interfaces and sub-λ objects generate THG → interface-selective contrast.

5. **Four-wave mixing (2ω₁ − ω₂):** Gouy phases partially cancel → works for bulk material → basis of CARS.

## Key Equations

| Equation | Name |
|----------|------|
| FWHM_nP = FWHM_1P / √n | Nonlinear resolution squeeze |
| d_1P = 0.61λ/NA | Single-photon resolution |
| d_2P = (0.61×2λ/NA)/√2 = 0.61√2·λ/NA | Two-photon resolution |
| SHG: L < λ₁/2 | Phase matching condition (from Gouy analysis) |
| THG: L < λ₁/3 | Stricter phase matching condition |

## Derivations

### Resolution Squeeze (2PEF example)

> **[RED BOX — know the derivation]**

Focal intensity: I(r) ∝ exp(−r²/σ²)  
2PEF signal: S ∝ I² ∝ exp(−2r²/σ²) = exp(−r²/(σ/√2)²)

New effective Gaussian has σ_eff = σ/√2 → FWHM_eff = FWHM/√2.

Combining with the 2× wavelength: d_2PEF = (0.61 × 2λ/NA) / √2 = 0.61√2 λ/NA ≈ 1.41 × 0.61λ/NA

So resolution is 1.41× worse than 1PEF — not 2× worse. The nonlinear squeeze partially compensates.

### Example (dye excited at 400 nm, NA = 1.2):

1PEF (400 nm): d = 0.61×400/1.2 = 203 nm  
2PEF (800 nm): d = 0.61×800/(1.2×√2) = 288 nm  
3PEF (1200 nm): d = 0.61×1200/(1.2×√3) = 352 nm  

> *In scattering tissue the 1PEF can't achieve 203 nm at any useful depth. 2PEF achieves 288 nm to 500 µm.*

## Physical Intuition

**Parametric vs. non-parametric:** A parametric process is instantaneous — photon in, photon out, no memory. Like a mirror. If the output photon has the wrong phase relative to its neighbours, they cancel (hence phase matching needed). A non-parametric process stores energy for picoseconds — the molecule "thinks about it" before emitting. The emitted photon has no phase relationship with the original photons → add incoherently (intensities, not amplitudes).

**Why THG is interface selective:** In bulk material, the contributions from each half of the focused beam (before and after focus) have opposite phases (due to Gouy shift) and cancel. Only when there's an inhomogeneity within the focal volume — an interface — does the cancellation fail.

## Common Mistakes

1. Saying 2PEF has the same or worse resolution than 1PEF (it has ~1.41× worse, not 2× worse)
2. Forgetting that parametric processes are coherent → phase matching needed
3. Saying SHG works for all materials (only non-centrosymmetric: collagen, myosin, quartz, BBO, etc.)
4. Thinking THG works from bulk material (it does NOT in a tight focus — only interfaces)
