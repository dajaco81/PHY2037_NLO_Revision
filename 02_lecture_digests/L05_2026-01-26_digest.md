# L05 Digest — Lorentz Oscillator Model and χ(ω)

## Core Concepts

1. **Lorentz oscillator model:** Model electrons in a dielectric as masses on springs with damping. Apply oscillating E field → solve ODE → get χ(ω).

2. **Linear restoring force → parabolic potential:** Valid for small displacements. Breaks down at high fields → nonlinearity. This approximation is the foundation of the Lorentz model and the reason we need perturbation theory later.

3. **χ(ω):** The electric susceptibility is complex and frequency-dependent. Real part → dispersion (refractive index). Imaginary part → absorption. Maximum at resonance.

4. **Polarisation timescales:**
   - Electronic: ~10⁻¹⁵ s (femtoseconds) → UV/visible
   - Atomic/vibrational: ~10⁻¹² s (picoseconds) → infrared
   - Orientational: ~10⁻⁶ s (microseconds) → microwave/far-IR
   
5. **Chemical specificity:** Electronic polarisation ≠ chemical specific (depends only on conjugation length). Atomic/vibrational polarisation = chemically specific (depends on specific atom masses and bond strength).

## Key Equations

| Equation | Name |
|----------|------|
| mṙ̈ + mγṙ + mω₀²r = qE₀e^{-iωt} | Lorentz equation of motion |
| ω₀ = √(k/m) | Resonant frequency |
| r₀ = (qE₀/m)/[(ω₀²−ω²)−iγω] | Displacement amplitude (solution) |
| χ⁽¹⁾(ω) = Nq²/(mε₀) / [(ω₀²−ω²)−iγω] | Linear electric susceptibility |
| ω_vib = (1/2π)√(k/μ_r) | Vibrational resonant frequency |
| μ_r = m₁m₂/(m₁+m₂) | Reduced mass |

## Derivations

### Lorentz Model → χ(ω)

1. Assume solution r = r₀e^{-iωt} (sinusoidal, same frequency as driving field)
2. Substitute: r̈ = −ω²r₀e^{-iωt}, ṙ = −iωr₀e^{-iωt}
3. Collect terms → r₀ = (qE₀/m)/D(ω) where D(ω) = (ω₀² − ω²) − iγω
4. Dipole moment: μ = qr₀; polarizability α = μ/E₀ = q²/(mD(ω))
5. Polarisation: P = Nαε₀E → χ⁽¹⁾ = Nq²/(mε₀D(ω))

## Physical Intuition

**Swing analogy:** Push someone at their natural frequency (resonance) → maximum amplitude, 90° phase lag. Push very slowly → small in-phase displacement. Push very quickly → no response (180° out of phase).

**Parabolic potential:** If restoring force is linear (F = −kx), potential is parabolic (U = ½kx²). This is the harmonic approximation. Works for small x; breaks down at large x where the real interatomic potential is anharmonic (like a Lennard-Jones potential).

**Why vibrational modes are chemically specific:** Each bond type (C–H, C=O, C–C) has a unique combination of masses and spring constant → unique resonant frequency → unique IR/Raman peak.

## Common Mistakes

1. Forgetting the imaginary term (−iγω) in the denominator D(ω) — it's essential for the damping/absorption
2. Confusing polarizability α (single atom) with susceptibility χ (bulk, per unit volume)
3. Using the wrong reduced mass formula: μ_r = m₁m₂/(m₁+m₂), not (m₁+m₂)
